# Screenshots Directory

Place your application screenshots here with the following naming convention:

- `home-dashboard.png` - Home dashboard screen
- `detection-flow.png` - Skin detection interface
- `mood-logging.png` - Mood logging screen
- `solace-interface.png` - Solace voice therapy interface
- `progress-view.png` - Progress tracking view

## Recommended Screenshot Specifications

- **Format:** PNG or JPG
- **Resolution:** Mobile device resolution (e.g., 1080x2340 for Android, 1125x2436 for iPhone)
- **Orientation:** Portrait mode
- **File Size:** < 1MB per image (optimize using tools like TinyPNG)

## Taking Screenshots

### Android (via Android Studio)
1. Run your app in the emulator or on a physical device
2. Navigate to the desired screen
3. Use the camera icon in Android Studio's device toolbar

### iOS (via Xcode)
1. Run your app in the simulator or on a physical device
2. Navigate to the desired screen
3. Use Cmd+S in the simulator or the screenshot button on physical device

### Optimization
Use image optimization tools to reduce file sizes:
- https://tinypng.com/
- https://squoosh.app/
- ImageOptim (macOS)
